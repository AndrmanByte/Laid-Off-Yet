# 示例：不想打工，能做什么小生意？

## 用户输入

「我不想再打工了，能做什么小生意？」

## Skill 输出要点（结构化）

1. **先稳住**：情绪可理解；小生意 **亏损与强度** 常被低估，需冷静算账而非对立「打工」。
2. **当前局面判断**：属于 **就业方向咨询** + **小生意/个体经营** 画像；必须问本金、家庭负担、能否承受亏损区间。
3. **现在最重要的事**：**最小成本验证** 优先于租铺、囤货、加盟（高风险）。
4. **今天/本周行动清单**：  
   - 今天：列固定成本与回本周期表（保守数字）。  
   - 本周：同城调研同类摊位/小店、访谈 2～3 个真实经营者；试跑最小闭环（预售/小批量/夜市试卖等合法形式）。
5. **可以说的话（家人）**：「我在用数据试跑，设了止损线 __，不是冲动辞职 all in。」
6. **风险提醒**：不夸大暴利；注意证照、食安、消防与合同；慎借贷重资产。
7. **时间线**：30 天内以验证为主；未验证前不建议断掉唯一现金流。
8. **行动收束**：若验证失败，列出 **可返回的打工/灵活就业** 兜底方向，不替用户选择「硬扛」。

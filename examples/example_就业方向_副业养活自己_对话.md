# 示例：想靠副业养活自己

## 用户输入

「我想做点副业养活自己。」

## Skill 输出要点（结构化 · 对齐 data/）

1. **先稳住**：「副业」到「养活自己」差的是 **稳定付费**；需验证与止损线。
2. **当前局面判断**：在 `role_profiles.json` 中重点看 `side_hustle_friendly: true` 的条目（如 `freelance_gigs`、`ecommerce_resell`、`self_media` 等），并对照 `accept_unstable_income` 与储蓄（`recommendation_rules`）。
3. **现在最重要的事**：若储蓄极紧，声明 **纯副业全职化风险**，建议并行合法兼职或最快到岗选项。
4. **今天/本周行动清单**：选一个 `id` 写清「谁付费、为什么、试单价」；7 天内完成一次最小成交或最小交付。
5. **可以说的话（客户）**：交付范围、价格、验收，书面留痕。
6. **风险提醒**：平台规则、税务合规、账期；勿借贷重库存。
7. **时间线**：7 天验证需求；30 天对照 `trial_plan_30d` 判断是否可达「养活自己」阈值。
8. **行动收束**：联动 `单干与副业.md`；输出各方向 **difficulty、recommendation_score、试水**；用户自选是否加码。

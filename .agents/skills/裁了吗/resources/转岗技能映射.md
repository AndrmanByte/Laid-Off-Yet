# 转岗技能映射（内容模板）

## 说明

- **权威数据**：`data/skill_mappings.json`  
  - `source_backgrounds`：常见原岗位/背景标签  
  - `mappings[]`：`from_background` → `target_role_id`，含 `transferable_skills`、`skill_gaps`、`transition_note`  
- 本文件用于人类阅读与 **无法读 JSON 时的 fallback**：向用户解释「你为什么能试这个方向、还差什么」。

## 使用方式

1. 从用户描述中归纳最接近的 `from_background`（可多选）。  
2. 在 JSON 中查找指向目标 `target_role_id` 的条目（同一背景可对应多条目标）。  
3. 输出时显式写出：**可迁移技能**、**缺口**、**转岗一句话建议**（`transition_note`）。

## 与 role_profiles 联动

选定方向后，将 `skill_mappings` 中的 gaps 与 `role_profiles.skills_to_build` 合并叙述，避免重复堆砌。

## 提醒

映射为 **启发式**，真实招聘以 JD 为准；跨行需 **作品/访谈/小规模试岗** 降低雇主风险感知。

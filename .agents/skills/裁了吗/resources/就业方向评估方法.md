# 就业方向评估方法（内容模板）

## 说明

- **规则数据**：`data/recommendation_rules.json`（`dimensions`、`rules`、`selection_policy`）。  
- **画像数据**：`data/role_profiles.json` 中的 `difficulty`、`recommendation_score` 等。

## 评估维度（摘要）

现金流紧急度、可迁移匹配、学习成本、岗位可得性、收入与稳定性权衡、身心负荷、30 天内可逆试水成本。细则见 JSON 内注释与 `dimensions`。

## 上手难度（1～5）与推荐指数（1～10）

- 来自 **`role_profiles.json`**，勿自行改分除非用户有个案强证据（须在回复中说明理由）。  
- **推荐指数**是在「典型失业/转型求助」语境下的**可落地程度**，不是行业优劣。  
- 规则引擎中的 `boost` / `penalize` 用于 **排序与过滤提示**，不是唯一答案。

## 收入区间声明

任何来自画像的收入表述必须附带：**城市线、公司规模、资历**差异极大，**以本地实时岗位为准**。

## Skill 禁止

- 保证「一定找到工作」或「一定达到某收入」  
- 用单一测试结果定终身职业  
- 忽视用户明确法律与伦理边界

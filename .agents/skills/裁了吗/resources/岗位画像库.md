# 岗位画像库（内容模板）

## 说明

- **权威数据**：仓库根目录 `data/role_profiles.json`（每条方向与 `data/career_paths.json` 的 `id` 对齐）。  
- 本文件提供 **字段模板** 与 **给 Agent 的阅读提示**：当无法解析 JSON 时，仍按下列字段口头输出。

## 单条方向画像字段（与 JSON 一致）

| 字段 | 说明 |
|------|------|
| id | 与 career_paths / role_profiles 一致 |
| name | 方向名称 |
| aliases | 别名/投递关键词 |
| category | `employment` 受雇白领/职能等 · `flexible` 灵活就业与小生意 · **`platform` 平台接单** · **`service` 服务业一线/门店** |
| suitable_backgrounds | 适合人群（非绝对） |
| **common_entry_barriers** | **常见进入门槛**（证照、车辆、健康证、平台审核、体力班次等） |
| common_job_titles | 常见招聘岗位名称 |
| job_requirements | 常见招聘要求 |
| core_skills | 核心技能要求 |
| transferable_skills | 从其他岗位可迁移技能 |
| skills_to_build | 需要补的技能 |
| difficulty | 上手难度 1～5 |
| recommendation_score | 推荐指数 1～10（可落地程度，非职业贵贱） |
| learning_cycle | 学习周期 |
| income_range | 收入大致范围（须声明仅供参考） |
| stability | 工作稳定性 |
| market_demand | 是否容易找工作（相对表述） |
| side_hustle_friendly | 是否容易做副业 |
| pros / cons | 优点 / 缺点 |
| trial_plan_30d | 30 天试水建议 |
| not_suitable_for | 不适合的人群 |

## 输出时注意

- **推荐指数**与**上手难度**需同时出现，并口头说明指数≠「职业好坏排名」。  
- **平台岗与服务业一线**须如实说明体力、班次、交通安全与平台罚款，不美化。  
- `income_range` 仅作区间参考，**以用户本地招聘软件/平台规则与实时单量为准**。
